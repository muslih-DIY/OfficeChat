# OfficeChat
![](icon-1.png)


## python version

Python 3.10.7

This application will help the office mate to communicate in the different topics.


    git clone https://github.com/muslih-DIY/OfficeChat.git
    
    cd OfficeChat
    
    python -m venv venv
    
    source venv/Scripts/activate
    
    pip install -r requirements.dev
    
    cd OfficeChat
    
    uvicorn apps:app
